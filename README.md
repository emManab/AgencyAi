🚀 Agentic Website

🌐 Live Demo: Visit Live Site

📌 Overview

This project is a modern frontend web application built using React + Vite, designed for speed, performance, and scalability. It includes a fully functional contact form powered by Web3Forms, enabling seamless form submissions without needing a backend.

The website focuses on delivering a clean UI/UX along with smooth performance and responsiveness across devices.

⚙️ Tech Stack
⚛️ React (Frontend Library)
⚡ Vite (Build Tool)
🎨 CSS / Tailwind (Styling)
📩 Web3Forms (Form Handling)
🌐 Netlify (Deployment)
✨ Features
⚡ Fast and optimized performance using Vite
🎯 Clean and modern UI/UX
📱 Fully responsive design
📩 Contact form integration (Web3Forms)
🚀 Deployed and accessible online
🧩 Component-based architecture

🧑‍💻 Getting Started
1️⃣ Clone the repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2️⃣ Install dependencies
npm install
3️⃣ Run development server
npm run dev
4️⃣ Build for production
npm run build
📩 Contact Form Setup (Web3Forms)
Go to Web3Forms
Get your Access Key
Add it to your form:
<input type="hidden" name="access_key" value="YOUR_ACCESS_KEY">
Submit form → receive responses directly in your email ✅
🚀 Deployment

This project is deployed using Netlify.

To deploy:

Push your code to GitHub
Connect repository with Netlify

Build command:

npm run build

Publish directory:

dist

Netlify provides instant deployment and scalability for modern frontend apps
